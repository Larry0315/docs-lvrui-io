常见的数据类型

Python中的数据类型可以分为两类，一类是 基本数据类型 一类是 引用数据类型

基本数据类型

  • 字符型： ‘a’, ‘message’
  • 数值型： 59, 1991.59
  • 布尔类型: True, False

引用数据类型

  • 列表： [1, 2, 3], [‘abc’, ‘xyz’]
  • 元组： (1, 2, 3), (‘abc’, ‘xyz’)
  • 字典： {‘name’=’larry’, ‘age’: 25}
  • 类

基本数据类型与引用数据类型的区别：前者传值，后者传址
